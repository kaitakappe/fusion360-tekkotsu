# 鉄骨作成支援アドイン - 使用方法

## 概要
このアドインは、Fusion 360上で鉄骨用のスプライスプレートとガセットプレートを簡単に作成できます。

## インストール済み機能

### 1. スプライスプレート作成
H形鋼用のスプライスプレートを自動生成します。

#### 対応プレートタイプ
- H形鋼用A1: 200×150×9mm
- H形鋼用A2: 250×150×9mm
- H形鋼用A3: 300×150×9mm
- H形鋼用A4: 350×150×9mm
- H形鋼用A5: 300×180×12mm
- H形鋼用A6: 350×180×12mm
- H形鋼用A7: 400×180×12mm

#### 使用方法
1. Fusion 360のツールバーから「スプライスプレート作成」をクリック
2. ドロップダウンメニューからプレートタイプを選択
3. 必要に応じて板厚とボルト穴径を調整
4. OKをクリックして生成

### 2. ガセットプレート作成
接合用のガセットプレートを自動生成します。

#### 対応プレートタイプ
- 取付5G1: 110×110×9mm（基本三角形）
- 取付7G52: 110×110×9mm（角切り）
- 取付9DG3: 140×140×9mm
- 取付10G4: 140×140×9mm（角切り）
- 取付17G5: 140×140×9mm
- 取付15G8: 170×170×9mm
- 取付7G57: 170×170×9mm（角切り）
- 取付20G8: 170×170×12mm
- 取付15DG10: 170×170×9mm

#### 使用方法
1. Fusion 360のツールバーから「ガセットプレート作成」をクリック
2. ドロップダウンメニューからプレートタイプを選択
3. 必要に応じて板厚とボルト穴径を調整
4. OKをクリックして生成

## 特徴
- **自動寸法設定**: プレートタイプを選択すると標準寸法が自動設定されます
- **カスタマイズ可能**: 板厚やボルト穴径を自由に調整できます
- **コンポーネント生成**: 各プレートは独立したコンポーネントとして作成されます
- **正確な寸法**: カタログの仕様に基づいた正確な3Dモデルを生成します

## アドインのリロード方法
Fusion 360でアドインを更新した場合：
1. Fusion 360で「ツール」→「アドイン」を開く
2. 「鉄骨作成支援」を選択
3. 「停止」→「実行」で再起動

## トラブルシューティング
- **コマンドが表示されない**: アドインを再起動してください
- **エラーメッセージが表示される**: Fusion 360のテキストコマンドウィンドウでエラー詳細を確認してください
- **プレートが正しく生成されない**: 寸法パラメーターを確認してください

## ファイル構造
```
鉄骨作成支援/
├── 鉄骨作成支援.py           # メインエントリーポイント
├── config.py                 # 設定ファイル
├── commands/                 # コマンドモジュール
│   ├── __init__.py          # コマンド管理
│   ├── splicePlate/         # スプライスプレートコマンド
│   │   ├── entry.py
│   │   └── __init__.py
│   └── gussetPlate/         # ガセットプレートコマンド
│       ├── entry.py
│       └── __init__.py
└── lib/                      # ユーティリティライブラリ
    └── fusionAddInUtils/
```

## 開発者向け情報
新しいプレートタイプを追加する場合：
1. 該当するentry.pyファイルを開く
2. プレートタイプ辞書（SPLICE_PLATE_TYPES or GUSSET_PLATE_TYPES）に新しいエントリを追加
3. 寸法、板厚、ボルト穴位置を指定
4. アドインを再起動
